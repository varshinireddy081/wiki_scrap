# wiki_scrap
Scrapping wiki pages and internal links
# Requirements 
python 3.10.0 (version 3.x)<br />
pip install request<br />
pip install bs4(BeautifulSoup)
# How to run                                                                                         
python wiki_scrap {wiki page name} {iterations} <br />
Example :python wiki_scrap https://en.wikipedia.org/wiki/MS_Dhoni 20
